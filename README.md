# Embedded Practice Repository

## 📂 Directory Structure
- `stm32/`
  - UART, SPI, I2C, ADC, FreeRTOS 실습
- `bsp/`
  - Bootloader, Device Driver 실습
- `qt/`
  - 기본 UI, 센서 모니터링 프로그램
- `opencv/`
  - 이미지 필터링, 얼굴 검출 실습
